== Description ==

This plugin was uploaded with the wrong name. It will disappear from the repository shortly and be replaced with one with a different name.

Easily embed Facebook posts into your pages via a simple shortcode in WordPress posts or pages.

Example useage:
`[facebookpost https://www.facebook.com/metronet/posts/10151735688640772]`


== Installation ==

Install and activate the plugin.

You can embed Facebook posts by adding the URL for the post to the facebookpost shortcode:
`[facebookpost https://www.facebook.com/metronet/posts/10151735688640772]`


== Frequently Asked Questions ==

= Why should I use this plugin? =

If you want to easily embed Facebook posts in your WordPress posts and pages

= Does it work for WordPress version x.x.x? =

We only provide support for the latest version of WordPress.
